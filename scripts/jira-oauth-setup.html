Jira OAuth 2.0 Setup (Offline)

Dieses Tool führt den OAuth-Flow lokal durch. Am Ende erhältst du einen Refresh Token, den du im Vault auf dem Server einträgst.

1. OAuth App Daten eingeben

Callback URL in der Atlassian Developer Console eintragen: http://localhost:19876/callback

2. Bei Atlassian autorisieren

Klicke den Button, autorisiere den Zugriff, und kopiere den code Parameter aus der Redirect-URL.

Der Browser wird auf http://localhost:19876/callback?code=XXXX weitergeleitet. Die Seite wird nicht laden (kein Server) — kopiere einfach den code= Wert aus der Adressleiste.

3. Token austauschen
